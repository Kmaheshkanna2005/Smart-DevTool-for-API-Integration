# Smart DevTool for API Integration

An AI-driven tool that extracts key endpoints and authentication methods from API documentation URLs to auto-generate ready-to-use wrapper classes.

## Project Phases
- [x] **Phase 1:** Research & Documentation Scraper (Current)
- [ ] **Phase 2:** AI Parsing & Schema Mapping
- [ ] **Phase 3:** Code Generation Engine
- [ ] **Phase 4:** Final UI & Polishing

## Setup
1. Clone the repo: `git clone https://github.com/Kmaheshkanna2005/Smart-API-Integrator.git`
2. Install dependencies: `pip install requests beautifulsoup4`
